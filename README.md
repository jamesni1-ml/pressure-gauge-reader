# Analog Pressure Gauge Reader

**Fully automatic analog pressure gauge reading using computer vision.**

YOLOv8n detects the gauge face → OpenCV corrects perspective from any camera angle → EasyOCR reads the scale numbers → classical CV detects the needle → math converts the angle to a pressure reading. Zero manual calibration — point at any gauge, get a reading.

## Architecture

```
Image → YOLOv8n Detection → Perspective Correction → EasyOCR Scale Reading → Needle Detection → Pressure Reading
         (~1-2 sec)            (~0.3 sec)              (~2-4 sec)             (~0.5 sec)          (instant)
```

**Training**: WSL Ubuntu, conda `vision-ml`, Python 3.10, PyTorch 2.6+cu124, Ultralytics  
**Deployment**: Raspberry Pi 4 (8GB), ONNX model via OpenCV DNN + EasyOCR (no PyTorch needed)

## Performance Target

- **400 images per hour** on Raspberry Pi 4 (8GB)
- ~4-7 seconds per image (9 second budget)
- Handles photos taken from different angles via perspective correction

## Quick Start

### 1. Training (WSL)

```bash
conda activate vision-ml
pip install easyocr

# Download dataset from Roboflow (YOLOv8 format) into dataset/
# Then open and run the notebook:
jupyter lab pressure_gauge_training.ipynb
```

### 2. Dataset

Go to [universe.roboflow.com](https://universe.roboflow.com), search **"pressure gauge"**, filter by **Object Detection**, pick a dataset with 500+ images covering diverse gauge types, download in **YOLOv8** format, and extract into `dataset/`.

### 3. Raspberry Pi Deployment

```bash
# Copy rpi_deploy/ folder to Pi
scp -r rpi_deploy/ pi@<PI_IP>:~/gauge_reader/

# On the Pi:
pip install -r requirements_pi.txt

# Batch process 400 photos
python3 gauge_reader.py --input ./photos/ --output results.csv

# Single image
python3 gauge_reader.py --single photo.jpg
```

See [rpi_deploy/README_PI.md](rpi_deploy/README_PI.md) for full Pi setup instructions.

## How It Works

1. **YOLOv8n** detects gauge face bounding boxes in the image
2. **Perspective correction** — if the gauge appears as an ellipse (off-angle photo), warps it to a circle
3. **EasyOCR** reads the numbers on the gauge scale and computes their angular positions
4. **Needle detection** — isolates the needle via color thresholding + Hough line detection, finds its angle
5. **Piecewise linear interpolation** maps the needle angle to a pressure value using the OCR-derived scale
